# Personal Expense Tracker (Multi-user)

**Author:** Atish Rana

**University:** Manipal University Jaipur

**Course:** BCA final year project

## Overview
A lightweight multi-user personal expense tracker built with Python, Tkinter, SQLite and Matplotlib. Users can register/login, add expenses, view and delete them, export to CSV, and visualize spending by category.

## Run
1. Install requirements: `pip install matplotlib`
2. Run: `python main.py`

## Files
- `main.py` - application code
- `expenses.db` - created automatically on first run (included)

## Notes
- Passwords are hashed with SHA-256 (sufficient for demo; for production, use bcrypt)
- This is a desktop app; no internet required
